# Project Documentation

## Overview
This repository contains the project files and documentation for the project.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/prcims/drcollazo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd drcollazo
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the project, run:
```bash
npm start
```
